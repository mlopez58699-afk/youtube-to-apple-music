# YouTube to Apple Music

A macOS tool that downloads YouTube playlists and imports them into Apple Music while preserving:

- Playlist order
- Song title
- Artist
- Album metadata
- Artwork

## Features

- Uses yt-dlp for downloading
- Uses Python for metadata handling
- Uses AppleScript to automate Apple Music

## Future Features

- Smart Mode
- Duplicate detection
- Automatic matching with existing Apple Music library
